# Documentation Assets

## Screenshots

To add the homepage screenshot:

1. Start the development server: `npm run dev`
2. Navigate to http://localhost:3000
3. Take a screenshot of the homepage
4. Save it as `screenshot-homepage.png` in this directory
5. The screenshot will automatically appear in the main README.md

**Recommended screenshot specifications:**
- Resolution: 1920x1080 or higher
- Format: PNG
- Include: Navigation bar, movie grid, and glassmorphism effects
- Browser: Chrome or Firefox (for best rendering)
